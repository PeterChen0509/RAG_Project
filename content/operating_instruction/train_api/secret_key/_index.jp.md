シークレットキーは「設定画面」で取得します。
作成済の予測モデルについて「API作成タブ」をクリックすると、この画面に遷移します。

「設定」をクリックすると、この画面に遷移します。
シークレットキー「作成」をクリックすると、シークレットキーを取得できます。
