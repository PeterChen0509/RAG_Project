**クラウド版**は、Google Chrome (Version 89 以降) / Microsoft Edge (Version 89 以降) で動作する、Web アプリケーションです。インストール不要で、Web ブラウザ上で動作します。また、データとモデルをチームで共有したり、モデル作成を複数同時に実行したりすることもできます。
